CSS Selectors

    Universal Selector will apply CSS to every element and its content in the documnet.

  • ID selctors
  • Class Selectors

    • Element selector, directly take element name and apply properties

  • Descedent selectors(use space) EX: div p{property :value;} --: here all the p under div will have the applied properties
    Ex:2 div a p{roperty:value} , property will be applied for app p, hich are under a, whic are under div
  • Sibling Combinator(use +), they are next to each other EX: p+h3{pro:val}
  • Child Combinator(use >) Ex:span>button{pro:val}
  • Attribute Selector
    Synta: element[type of attribute=attribute value]{pro:val}
    EX: input[type="text"]{ color:red }
