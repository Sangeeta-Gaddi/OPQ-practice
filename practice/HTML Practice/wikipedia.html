HTML

HTML (HyperText Markup Language) is the standard markup language used to create web pages.
It describes the structure of a web page using elements and tags.
HTML is used together with other technologies such as CSS and JavaScript.

Overview and Uses of HTML

HTML was created to allow documents to be shared on the World Wide Web. It uses tags to define elements like headings, paragraphs, links, images, and lists. Web browsers read HTML files and display them as web pages. HTML (HyperText Markup Language) is the foundational standard markup language used to create and structure content on the World Wide Web. It is not a programming language but a system of using tags to annotate text, images, and other multimedia, instructing a web browser on how to display the content. Think of HTML as the "skeleton" of a webpage, providing the organizational framework that other technologies, like Cascading Style Sheets (CSS) for visual styling and JavaScript for interactivity, build upon.

HTML is used to:

  • Create web pages
  • Structure content on the internet
  • Link documents together using hyperlinks

History

HTML was first developed by Tim Berners-Lee in the early 1990s. It has gone through many versions over the years. The latest version is called HTML5.

Common HTML Elements

  • <html> : Defines the root of an HTML document.
  • <head> : Contains metadata such as title and character encoding.
  • <title> : Sets the title of the web page shown in the browser tab.
  • <body> : Contains the visible content of the web page.
  • <h1> to <h6> : Used to define headings, from most important (<h1>) to least important (<h6>).
  • <p> : Defines a paragraph of text.
  • <ul> : Defines an unordered (bulleted) list.

Basic Structure

This is Basic HTML structure shows a heading, a paragraph, and a link.

<!DOCTYPE html>
<html>
<head>
    <title>My Web Page</title>
</head>
<body>

    <p>This is a paragraph.</p>

    <a href="https://www.example.com">This is a link</a>

</body>
</html>

Points To Remember

  • HTML stands for: HyperText Markup Language.
  • Structure: HTML documents have <html>, <head>, and <body> sections.
  • Paragraphs: Use <p> to define paragraphs.
  • Headings: Use <h1> to <h6> for headings.
  • Links: Use <a href="URL"> to create hyperlinks.
  • Images: Use <img src="URL" alt="description"> to display images.
